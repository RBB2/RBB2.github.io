Here are some of the projects I have done with my cozmo robot.

Find a Ball

Here is a demonstration of the cozmo robot running my code to search for a black ball. Once the robot has detected a black ball, it is programmed to drive towards the ball and hit the ball with the fork lift once it is close enough. The code first takes an image from the robotics camera, then it applies a guassian blur, before using openCV's Hough transform function to find a circle on the screen. The mapping the robots wheels to the location of the circle the robot will drive towards the ball.
